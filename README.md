# CloudStream Provider — 1jour1film (metadata only)

This repository contains a CloudStream provider plugin (metadata-only) for https://1jour1film1125.site/.

⚠️ This plugin does **not** provide or extract streaming links — it only collects public metadata (titles, posters, descriptions, tags).

## Installation

1. Copy `provider/OneJour1FilmProvider.kt` into your CloudStream providers package.
2. Rebuild CloudStream with Gradle (`./gradlew assembleDebug`).
3. Install and enjoy browsing metadata in the app.
